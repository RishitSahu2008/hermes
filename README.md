# hermes
This is a project using an onboard senson(in my case a dht11 sensor). This sensor will share the data to the microcontroller and will be displayed on a small oled display. 
The main reason I am making this project is -
1) Its my first project like this where i have to document each and everything so I chose an easy project.
2) I want to learn about the XIAO RP2040 microcontroller.
3) Also I have the DHT 11 sensor and oled display already with me. So the project will not be too costly .

I have not designed a pcb for the project at it is expensive rather i can invest some time of mine and make it on a cheap board myself.
